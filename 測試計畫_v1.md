# v0.7 數值合理性驗證與調整（2026-06-16）

> 方法：以 Node 寫了對齊 `battle.js` 的離散時間戰鬥模擬器（atkSeq 循環、DEF 減傷、出手間隔、技能 CD/次數、法師 AoE、敵人前排權重、治療門檻、暈/暴/連擊），每情境跑 500–600 次取勝率與殘血；另做經濟與聲望可達性驗算。
