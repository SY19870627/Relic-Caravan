# 法師職業規劃器 — 發想 v0.1（簡單草稿）

> 最簡發想，待之後比照《戰士職業規劃器_設計稿》正式展開（技能效果／點數／公式）。

## 一句話定位
後排範圍魔法，一次掃蕩成群敵人。改版後以**法術流派**選「穿透真傷」或「純爆發高傷」。

## 招牌系統：法術
遇敵時依分支以對應法術系作戰，決定傷害的「質」（穿透）或「量」（爆發）。

## 大招（法術，二選一裝 1）
- **靈魂魔法（無視防禦）**：真傷／穿透的代表大法術，對高 DEF 敵人與王特別有效。
- **傷害魔法（高傷害）**：大範圍爆發的代表大法術，對付成群敵人。

（靈魂／傷害的相關法術成為自由混搭的主動小招／被動，不再鎖系列。）

## 種子技能點子
- 靈魂線：靈魂震爆(無視 DEF 單體)、心靈撕裂(無視 DEF 持續)、禁錮(暈)、汲魂(傷害回血)
- 傷害線：火球(AoE 高傷)、連環施法(多段)、過載(自身傷害暴漲一段時間)、隕石(大範圍爆發)
- 共用被動：奧術連鎖(攻擊濺射)、法力護盾(開場護盾)

## 與規劃器框架的關係
共用全套框架：職業點數上限、**1 大招＋3 主動小招＋被動（免費不佔槽、不限數）、統一池自由混搭**、Lv1–20 配置、免洗、地城內自動發放、可重置重配。

**可複用的引擎**：法師本身已 `aoe:true`；引擎已有 `pierce`（無視防禦）與 `aoeBonus`（奧術連鎖）。靈魂魔法＝`pierce` 拉到 100%／真傷；傷害魔法＝沿用 AoE 並加大倍率。改動相對小。
